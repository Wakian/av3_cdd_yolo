# Imagens reais capturadas pela equipe

Coloque aqui as fotos que vocês mesmos vão tirar fazendo letras em LIBRAS — essas fotos são exigência do projeto (item 4 da estrutura: *"Capturar uma foto ou vídeo original produzido pelos próprios integrantes do grupo"*).

## Dicas para boas capturas

- **Fundo:** neutro (parede branca, lousa, papelão) — evita falsos positivos
- **Iluminação:** frontal, uniforme, sem sombras fortes
- **Distância:** 30–80 cm da câmera
- **Enquadramento:** mão centralizada, visível por inteiro, sem cortes
- **Formato:** JPG ou PNG, resolução mínima 640×640
- **Variedade:** capture pelo menos 5–10 letras diferentes, com integrantes diferentes se possível, para validar generalização

## Como usar

1. Tire as fotos com o celular.
2. Transfira para `real_world_test/images/`.
3. No notebook (Colab), faça upload dessas mesmas fotos quando a célula `files.upload()` rodar.
4. As predições anotadas devem ser salvas em `../results/predicoes_reais.png`.
